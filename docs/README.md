# E2E Verifier Documentation

This directory contains the comprehensive documentation for the E2E Verifier tool.

## 📖 Viewing the Documentation

### Local Preview
Simply open `index.html` in your web browser:

```bash
# Using default browser
open docs/index.html

# Or with a simple HTTP server
npx http-server docs -p 8080
# Then visit http://localhost:8080
```

### Features
- **🎨 Dark theme** with professional GitHub-inspired design
- **📱 Fully responsive** - works on desktop, tablet, and mobile
- **🔍 Interactive navigation** with smooth scrolling and active section highlighting
- **📋 Copy functionality** on all code blocks and command examples
- **🎯 Complete coverage** of CLI commands, configuration, API usage, and more
